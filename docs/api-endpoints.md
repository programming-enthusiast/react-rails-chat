# API Endpoints

## HTML API

### Root

- `GET /` - loads disclone web application

## JSON API

### Users

- `POST /api/users`
- `GET /api/users/:id`

### Session

- `POST /api/session`
- `DELETE /api/session`

### Servers

- `POST /api/servers`
- `GET /api/servers/:id`
- `DELETE /api/servers/:id`
- `PATCH /api/servers/:id`

### Channels

- `POST /api/channels`
- `PATCH /api/channels/:id`
- `DELETE /api/channels/:id`
- `GET /api/channels/:id`

### Messages 

- `POST /api/messages`
